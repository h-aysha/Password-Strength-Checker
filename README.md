# Password-Strength-Checker
A C++ project that checks the strength of password based on character diversity and length.

This is a simple c++ code to check the strength of a password. 
It evaluates the strength based on character diversity and length 
of the password and gives one of the following output:
-WEAK
-MODERATE 
-STRONG
This project demonstrates basics of Object Oriented Programming(OOP) concepts such as classes.
